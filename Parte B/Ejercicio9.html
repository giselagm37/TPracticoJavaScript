PREGUNTA 1 RESPUESTA: VERDADERO
let: Se utiliza para declarar variables con ámbito de bloque. Puede ser reasignada. const: Se utiliza para declarar constantes con ámbito de bloque. No puede ser reasignada después de la inicialización, pero el contenido de un objeto o matriz declarado con const puede ser modificado. var: Se utiliza para declarar variables con ámbito de función. Es más antigua y tiene comportamientos diferentes en comparación con let y const. No tiene ámbito de bloque y puede ser reasignada. Entender estas diferencias es esencial para escribir código moderno y mantener un buen control sobre el alcance y la mutabilidad de las variables en JavaScript.

PREGUNTA 2 RESPUESTA: FALSO
JavaScript es conocido por ser un lenguaje de programación de tipado débil o dinámico. Esto significa que las variables en JavaScript no están fuertemente ligadas a un tipo de dato específico y pueden cambiar de tipo durante la ejecución del programa.
